# dioxusmusic

a music player built by dioxus and rust.

```
project/
├─ assets/ # Any assets that are used by the app should be placed here
├─ src/
│  ├─ main.rs # main.rs is the entry point to your application and currently contains all components for the app
├─ Cargo.toml # The Cargo.toml file defines the dependencies and feature flags for your project
```

## Features to implement

- [ ] Play local music files
- [ ] Control music playback (play, pause, stop, seek)
- [ ] Display current track information (title, artist, album cover)
- [ ] Create and manage playlists
- [ ] Save and load playlists
- [ ] Add support for web music from cloud services (e.g. aliyun, webdav, Nextcloud, etc.)
